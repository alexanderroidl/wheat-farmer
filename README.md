Poppy Play
===

# Inhalte
* Abhängigkeiten
* Installation
* Benutzung

# Abhängigkeiten
* [Node.js](https://nodejs.org)

# Installation
```bash
npm install # Node-Pakete installieren
```

# Benutzung
```bash
npm run build # Einmal kompilieren
npm run dev # Einmal kompilieren und anschließend watchen
```
